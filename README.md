# PieMenuEditor for Blender5+
A fork of Pie Menu Editor, now ready for blender 5.0+

#ReadMe

The legendary Pie Menu Editor Blender Add-on, a core extension for many workflows, is no longer maintained by its original creator and stopped working after Blender 4.5.
It's been updated to be compatible with the latest Blender 5.0+

All the credits go to the original creator: https://roaoao.gumroad.com/l/pie_menu_editor
Sadly, I couldn't find Roaoao's contact. But this fork is permitted under the original General Public License 2.0

This fork of the original addon aims to keep the original functionality intact for future Blender versions, as long as I'm still using this addon.

Please submit a ticket if you have encountered any issues!
